
خبرزات — بستهٔ استاتیک سایت خبری
----------------------------------

این بسته شامل یک سایت استاتیک نمونه (فرانت‌اند) با قابلیت‌های زیر است:

- صفحهٔ اصلی (index.html) که اخبار را از articles.json می‌خواند و به‌صورت پویا رندر می‌کند.
- قالب صفحهٔ خبر (article.html) که بر اساس query string (?slug=) خبر را نشان می‌دهد.
- فایل استایل (styles.css) و اسکریپت (script.js) با امکانات جستجو، فیلتر و صفحه‌بندی.
- تصویر پیش‌فرض /images/ii.png که در متادیتا برای شبکه‌های اجتماعی استفاده می‌شود.
- sitemap.xml و robots.txt و manifest.json برای پیکربندی اولیه SEO و PWA.
- articles.json شامل 60 خبر نمونه برای آپلود مستقیم روی سرور.

طریقهٔ استفاده:
1. محتوای پوشه را در روت سرور استاتیک خود قرار دهید (مثلاً /var/www/html یا سرویس‌هایی مانند Netlify/Vercel).
2. فایل images/ii.png را در مسیر /images/ii.png نگه دارید (همین نام لازم است برای نمایش تصویر در متا).
3. برای هر خبر جدید، یک آبجکت در articles.json اضافه کنید و یک فایل article-specific در صورت نیاز بسازید.
4. برای SEO بهتر، پیشنهاد می‌شود sitemap.xml را با آدرس‌های دقیق سایت واقعی به‌روزرسانی کنید و فایل‌های صفحات داخلی را سرور-ساید رندر کنید (SSR) یا HTML ایستا برای هر خبر تولید کنید.
